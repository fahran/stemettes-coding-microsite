Tools

To start programming Python, you'll need a few bits and pieces.

  1. The Python 2.7 Interpreter allows your computer to understand the Python language. Get the 32-bit version.
  2. Eclipse IDE (Inregrated Development Environment lets you run code, debug it to find problems, and manage your projects.
  3. Make sure you have Java installed - Eclipse needs it to run.
  4. The Pydev Plugin for Eclipse gives Eclipse a greater understanding of Python. Follow the instructions from "Installing with the update site".
  5. The Pygame package gives Python the power to draw. Get the version for Python 2.7.
