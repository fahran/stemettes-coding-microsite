Lesson 9: Classes and Objects

The __init__ method is used to create a new instance of an object. Properties of the object like "height" and "amount" are defined here.

Other methods within the class can be called on an instance of the object, eg. bottle.drink(250).

This is object-oriented coding, and a very useful way of dividing up your work.

def run(bottle): bottle.drink(250) return bottle.amount class Bottle: def __init__(self, liquid): self.amount= 500 self.height = 10 self.empty = False self.liquid = liquid def amount_left(self): return self.amount def drink(self, amount): self.amount -= amount if self.amount %lt;= 0: self.empty = True
